This is a program which uses M*N worker threads to compute the product of two matrices A and B. 
Compile using 'make'.

Usage: ./matrix_multiply [optional filename for matrix A] [optional filename for matrix B]

If only one filename/argument is given, default matrix B from the book will be used for matrix B.
If no filenames/arguments are given, default matrices A and B from the book will be used for both matrices.

Test matrices are provided for the user.
